# 🧮 NeoCalc - Modern Calculator

A sleek, feature-rich calculator web application with support for all basic arithmetic operations, keyboard controls, and a modern glass-morphism design.

## 🌟 Live Demo

Experience the calculator live: **[https://chaithanya8861.github.io/CodeAlpha_Calculator/](https://chaithanya8861.github.io/CodeAlpha_Calculator/)**

## ✨ Features

### Core Functionality
- **Basic Arithmetic Operations** – Addition (+), Subtraction (−), Multiplication (×), and Division (÷)
- **Real-time Display** – Shows current input and previous operation
- **Clear Functions** – AC (Clear All) and Delete (⌫) buttons
- **Decimal Support** – Handles floating-point numbers accurately
- **Error Handling** – Prevents division by zero and invalid operations

### 🎯 Bonus Features
- **⌨️ Full Keyboard Support** – Use your keyboard for faster calculations
  - Number keys (0-9)
  - Operators (+, -, *, /)
  - Enter or = for equals
  - Backspace for delete
  - Escape for clear all
- **🎨 Modern UI Design**
  - Glass-morphism effect with backdrop blur
  - Smooth 3D button press animations
  - Gradient backgrounds
  - Hover effects for better interactivity
- **📱 Responsive Design** – Works perfectly on desktop, tablet, and mobile devices

## 🛠️ Technologies Used

- **HTML5** – Semantic structure
- **CSS3** – Modern styling with Flexbox/Grid, gradients, and animations
- **Vanilla JavaScript** – Core calculator logic and event handling

## 🚀 How to Use

### Mouse/Touch Controls
1. Click number buttons (0-9) to enter digits
2. Click operator buttons (+, −, ×, ÷) for calculations
3. Click `=` to see the result
4. Click `AC` to clear everything
5. Click `⌫` to delete the last digit

### Keyboard Controls
| Key | Action |
|-----|--------|
| `0-9` | Enter numbers |
| `.` | Add decimal point |
| `+` `-` `*` `/` | Select operation |
| `Enter` or `=` | Calculate result |
| `Backspace` | Delete last character |
| `Escape` or `Delete` | Clear all (AC) |
